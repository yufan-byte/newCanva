# Infinite Canvas 本地素材导入 Chrome 插件

用于扫描当前网页图片，批量导入到素材库管理里的「本地素材」。

## 安装

1. 打开 Chrome：`chrome://extensions/`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本目录：`tools/chrome-local-asset-importer`

## 使用

1. 先启动 Infinite Canvas 后端服务。
2. 点击浏览器右上角插件图标。
3. 输入服务地址，例如 `127.0.0.1:8767`、`192.168.1.10:3000`，也可以填写完整地址 `http://192.168.1.10:3000`。
4. 点击「连接」。
5. 可点击图钉按钮打开 Chrome 侧边栏，侧边栏不会因为切换页面而关闭。
6. 在任意网页点击「扫描」，插件会列出当前页面可访问的图片。
7. 勾选图片，选择平台和模型。
8. 可开启「导入后智能分类」，点击「导入所选」。
9. 也可以点「下载到本地」把所选素材直接存到浏览器下载目录——**这一步不需要连接后端**，可当作独立采集工具使用。选多张时会自动打包成一个 zip 压缩包（以「文件夹」命名），单张则直接下原文件。

导入后的文件会写入：

```text
assets/uploads/<你填写的文件夹>
```

素材库管理页面的「本地素材」会同步显示这些图片；智能分类结果会写入同名 `.classification.json`。

## 支持的素材来源

- 普通 `http/https` 图片、视频（由后端下载）。
- `blob:` 资源（如飞书文档）：在网页上下文里读取字节后以 base64 上传。
- `data:` 内联图片、SVG `<image xlink:href>`（如夸克文档整页渲染）。
- 跨域 `<iframe>` 内的素材（扫描与读取均覆盖所有 frame）。
- `<canvas>` 画布内容（如腾讯文档、网页版 PDF、图表）：用 `toDataURL` 导出；若画布被跨域图片污染会抓取失败，属正常现象。
- 勾选「扫描前自动滚动加载」可触发长文档 / 懒加载页面把图片渲染出来再扫描。
- 「截取当前可见画面」按钮：用浏览器合成的屏幕画面截图，适用于**地图 / WebGL / 加密画布**等 `toDataURL` 读出来是黑屏的内容（只截可见视口）。
- **视频**：扫描会收 `<video>`、`source` 以及 `.mp4/.webm/.mov/.m4v/.flv` 直链；后台还会**嗅探网络请求**里通过 XHR/fetch 加载的媒体（DOM 里看不到的），合并进扫描结果。
  - 直链文件（mp4/webm/flv 等）可直接导入下载。
  - 标记为「M3U8 / 流」的是 HLS/DASH 流媒体清单，由"播放列表+大量切片"组成，**本工具不下载合流**，需用专门的下载器处理（清单 URL 已为你列出）。
  - blob/MSE 流媒体（B站/YouTube 等）和 DRM 加密视频（Netflix/会员视频）无法通过直链获取。

## 注意

- DRM 加密视频（Netflix 等）、WebGL/Figma 实时渲染、闭合 shadow DOM 无法抓取。
- 局域网使用时，Infinite Canvas 服务需要绑定到局域网可访问的地址，并确保防火墙允许访问对应端口。
- 图钉按钮会打开 Chrome 侧边栏；扫描时会读取当前浏览器窗口正在激活的网页标签页。
- 有些网站会限制防盗链，后端下载失败时插件会显示失败数量。
- 智能分类会调用你选择的平台和模型，请确认 API Key 已在 API 设置中配置。
